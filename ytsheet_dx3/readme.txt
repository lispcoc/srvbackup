##################################
########### 設置の手順 ###########
 1. config.cgiを開いて、管理用パスなどを設定する
 2. ゆとシートのファイルをすべてサーバーにアップする
 3. 下記の表に従ってパーミッションを設定する
 4. ブラウザからシートのTOPページ(http://○○○/ytsheet_dx3/)へアクセスして
    問題なく表示されれば設置完了


##################################
#### パーミッション(属性)設定 ####
 (※一般的なものです。サーバーによっては違うので注意)

 /ytsheet_dx3 <707> カレントディレクトリ(名前は変更可)
  |
  |- index.cgi     <705> 基本CGI
  |- admin.cgi     <705> 管理用
  |- color.cgi     <705> キャラシート色セット表示CGI
  |- colorset.cgi  <604> キャラシート色セットデータ
  |- config.cgi    <604> 設定ファイル
  |- setting.cgi   <604> 設定ファイル2 (流派など追加したい人のみ変更)
  |- file.cgi      <604> 設定ファイル3 (基本いじりません)
  |- charpass.cgi  <606> パスワードファイル
  |- charliss.cgi  <606> 一覧データファイル
  |- help.html     <604> 取扱説明書
  |- style.css     <604> キャラ一覧のCSS
  |
  |-/css <705> CSSファイル収納ディレクトリ
  |  |
  |  |- chara.css  <604> キャラシートのCSS(基本)
  |  |- edit.css   <604> 編集画面用CSS
  
  |-/lib <705> ライブラリ格納ディレクトリ
  |  |
  |  |- 〜.pl, .js <604> ライブラリ群
  |
  |-/data   <707> キャラクターデータ収納ディレクトリ
  |-/backup <707> バックアップ収納ディレクトリ
  |
  |-/tmp    <707> 画像収納ディレクトリ
     |
     |- !none.jpg <604> ブランク画像
